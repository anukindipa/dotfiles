# css files for firefox

Based on Simple fox: https://github.com/migueravila/simplefox 

and tabs on bottom: https://github.com/MrOtherGuy/firefox-csshacks/tree/master/chrome/tabs_on_bottom_v2.css

# usage

go to C:\Users\<USERNAME>\AppData\Roaming\Mozilla\Firefox\Profiles\XXXXXXX.default-XXXXXX
and paste `userChrome.css`, `userContent.css` and `tabs_below_search.css`
